# 西南大学物联智能创新产业化中心
框架集成：spring+springMVC+mybatis
* 完成系统基础框架整个流程

# 项目结构
* 1.dto ：数据实体层，映射数据库
* 2.dao : 数据访问层，访问数据库
* 3.service : 业务逻辑层接口
    serviceImpl : 业务逻辑层实现
* 4.controller ：业务控制层
* 5.webapp : view 层
* 6.util : 工具包
